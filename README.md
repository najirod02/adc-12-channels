# ADC-12-channels
A simple project to read 12 analog signals and printing their digital values into UART.

## Pin used
We are using ADC2, in particular starting from channel IN0 to channels IN14 which corresponds to:
- IN0 $\rightarrow$ PA0
- IN1 $\rightarrow$ PA1
- IN4 $\rightarrow$ PA4
- IN6 $\rightarrow$ PA6
- IN7 $\rightarrow$ PA7
- IN8 $\rightarrow$ PB0
- IN9 $\rightarrow$ PB1
- IN10 $\rightarrow$ PC0
- IN11 $\rightarrow$ PC1
- IN12 $\rightarrow$ PC2
- IN13 $\rightarrow$ PC3
- IN14 $\rightarrow$ PC4
